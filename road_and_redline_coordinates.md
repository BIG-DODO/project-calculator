## 四、CAD 脚本使用说明

**文件位置**：`C:\Users\Admin\.kimi_openclaw\workspace\project-calculator\road_and_redline.lsp`

### 加载步骤
1. 打开 AutoCAD
2. 命令行输入：`AP`（加载应用程序）
3. 选择文件 `road_and_redline.lsp` → 点击 **加载**

### 执行命令

| 命令 | 功能 |
|:---:|:---|
| `DRAW_REDLINE` | 只绘制项目红线 |
| `DRAW_ROAD` | 只绘制道路范围 |
| `DRAW_ALL` | 同时绘制红线 + 道路 |

### 绘制效果
- **红线**：红色闭合多段线 + 黄色点号标注 + 面积标注
- **道路**：青色闭合多段线 + 道路名称标注 + 面积标注

## 五、调整道路宽度

如需修改道路宽度（当前 25m），编辑脚本中以下数值：

```lisp
; 在 road_and_redline.lsp 中，修改这 6 个偏移值：
(-10818.231  8037.941)    ; J8 向南偏移：原 Y 8062.941 - 25 = 8037.941
(-10898.883  8041.043)    ; J9 向南偏移：原 Y 8066.043 - 25 = 8041.043
(-10900.728  8041.104)    ; J10 向南偏移：原 Y 8066.104 - 25 = 8041.104
(-10898.580  8098.279)    ; J1 向南偏移：原 Y 8123.279 - 25 = 8098.279
(-10873.580  8123.279)    ; J1 向东偏移：原 X -10898.580 + 25 = -10873.580
(-10662.453  8115.266)    ; J2 向东偏移：原 X -10687.453 + 25 = -10662.453
```

将 `25` 替换为实际道路宽度，重新计算偏移值即可。
