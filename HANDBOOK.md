## 三、计算逻辑详解（核心公式）

> 以下公式按 `calculate()` 函数执行顺序排列

### 3.1 辅助函数

```javascript
function getBuildingDensity(far) {
  if (far < 1.5) return 0.45;  // 容积率 < 1.5，建筑密度 45%
  if (far < 2.0) return 0.42;  // 1.5 ≤ 容积率 < 2.0，建筑密度 42%
  return 0.40;                   // 容积率 ≥ 2.0，建筑密度 40%
}

function isHangzhou(region) {
  return region === '杭州';      // 杭州区域特殊处理地下面积
}
```

### 3.2 基础面积计算

```
地上建筑面积（aboveGroundArea）= 用地面积 × 容积率
绿地面积（greenArea）= 用地面积 × 绿地率
建筑基底面积（buildingBaseArea）= 用地面积 × 建筑密度
道路面积（roadArea）= 用地面积 × 30%（固定比例）
```

### 3.3 机动车停车位

```
厂房部分面积 = 地上建筑面积 × (1 - 配套用房占比)
配套部分面积 = 地上建筑面积 × 配套用房占比

机动车总车位（totalVehicle）= 
  ceil(厂房部分面积 × 厂房配建指标 / 100) + 
  ceil(配套部分面积 × 配套配建指标 / 100)
```

> 注：原文为简化为一步计算，等效公式。

### 3.4 非机动车停车位

```
非机动车总车位（totalNonVehicle）= ceil(地上建筑面积 × 1.0 / 100)

分配规则：
  if totalNonVehicle ≤ 500:
    groundNonVehicle = totalNonVehicle
    undergroundNonVehicle = 0
  else:
    groundNonVehicle = ceil(totalNonVehicle / 2)
    undergroundNonVehicle = floor(totalNonVehicle / 2)
```

### 3.5 地面机动车停车位（关键逻辑）

```
非机动车位占地面面积 = groundNonVehicle × 1.5（每车位1.5m²）

可用停车面积（availableParkingArea）= 
  用地面积 - 道路面积 - 绿地面积 - 建筑基底面积 - 非机动车位地面面积

地面机动车位（groundVehicle）= floor(availableParkingArea / 35)

// 兜底：不足10个则归0，剩余全转入地下
if groundVehicle < 10:
  groundVehicle = 0
```

> **参数演进**：50 → 25 → 35（最终定值）。
> - 50：原始预留值，停车位过于宽裕
> - 25：优化版，停车位紧凑
> - 35：平衡版，兼顾实际可用面积与停车效率（当前定值）

### 3.6 地下机动车停车位

```
地下机动车位（undergroundVehicle）= totalVehicle - groundVehicle
```

### 3.7 地下建筑面积

```
// 区域差异化：杭州 vs 其他
if 区域 == '杭州':
  地下面积 = undergroundVehicle × 42 + undergroundNonVehicle × 1.5
else:
  地下面积 = undergroundVehicle × 35 + undergroundNonVehicle × 1.5
```

> 杭州地下机动车位单辆面积 42m²，其他城市 35m²。非机动统一 1.5m²/位。

### 3.8 汇总输出

```
总建筑面积 = 地上建筑面积 + 地下建筑面积
亩数 = 用地面积 / 666.7
```

| 输入项 | 类型 | 必填 | 说明 |

| 融资成本率 | number | 是 | 年化利率，如 6.5% |
| 建设周期 | number | 是 | 月 |
| 出租率/去化率 | number | 是 | 分阶段输入或平均率 |

### 7.2 静态财务模型（即时计算输出）

| 输出项 | 公式 |
|---|---|
| 项目总投资 | 土地成本 + 建安成本 × 面积 + 其他费用 |
| 厂房年租金收入 | 厂房面积 × 租金 × 出租率 × 12 |
| 配套年租金收入 | 配套面积 × 租金 × 出租率 × 12 |
| 总年收入 | 厂房租金 + 配套租金 |
| 静态投资回收期 | 总投资 / 年净收益 |
| 静态投资回报率 | 年净收益 / 总投资 |

### 7.3 动态财务模型（全周期现金流模拟）

| 输出项 | 说明 |
|---|---|
| 逐年现金流表 | 建设期（支出）→ 运营期（收入-支出） |
| 净现值（NPV）| 按折现率计算项目全周期净现值 |
| 内部收益率（IRR）| 使 NPV = 0 的折现率 |
| 动态投资回收期 | 考虑折现后的累计现金流回正年份 |
| 敏感性分析 | 租金波动 ±10% / 20% 对 IRR 的影响 |

### 7.4 可视化输出（建议）

- **图表**：现金流曲线图、投资回报对比图、敏感性蜘蛛图
- **表格**：逐年现金流明细表、季度/月度收入预测表
